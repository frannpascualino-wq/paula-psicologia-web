# Paula Psicología — Web

Web profesional para el consultorio de Paula, psicóloga clínica. Stack: React + Vite + Tailwind CSS.

## Skills disponibles

Este proyecto tiene skills locales configuradas en `.claude/skills/`. Invocarlas con `/nombre-del-skill` al inicio de la tarea.

| Skill | Cuándo usarlo |
|-------|---------------|
| `/frontend-design` | Construir o modificar componentes visuales, secciones, layouts, animaciones, responsividad. |
| `/brand-guidelines` | Decisiones de identidad visual: paleta, tipografía, tono de voz, iconografía, fotografía. |
| `/canvas-design` | Estructura de páginas, flujo de secciones, jerarquía visual general, arquitectura de la UX. |
| `/doc-coauthoring` | Redactar o editar textos del sitio: "Sobre mí", servicios, FAQ, artículos de blog. |
| `/internal-comms` | Comunicación profesional del proyecto: resúmenes de avance, briefings, feedback a proveedores. |
| `/claude-api` | Integrar la API de Anthropic/Claude: chatbot, formulario inteligente, generación de contenido. |

## Stack técnico

- **Framework:** React 18 + Vite
- **Estilos:** Tailwind CSS
- **Carpeta de componentes:** `src/components/`
- **Carpeta de datos/contenido:** `src/data/`

## Convenciones

- Componentes en PascalCase, archivos en kebab-case.
- Sin comentarios que expliquen qué hace el código — los nombres lo dicen.
- Mobile-first siempre. Probar desde 320px.
- No inventar contenido sobre Paula sin confirmación — marcar con `[REVISAR: ...]`.

## Objetivo del sitio

Que los pacientes potenciales lleguen, entiendan quién es Paula y qué hace, y contacten para una primera consulta. Cada decisión de diseño y contenido debe servir a ese objetivo.
